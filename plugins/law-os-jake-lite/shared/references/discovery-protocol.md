# Discovery Response Protocol

## 4-Step Response Format

1. **Scope + Definitions:** Confirm request scope and identify ambiguities.
2. **Objection Analysis:** Apply objection categories A-J.
3. **Response Position:** State produce / withhold / partial with reason.
4. **Production Plan:** Define documents, custodians, and production timing.

## Objection Categories (A-J)

A. Relevance
B. Overbreadth
C. Undue burden
D. Privilege / work product
E. Vagueness / ambiguity
F. Compound request
G. Assumes facts not in evidence
H. Duplicative / already produced
I. Confidentiality / protective order needed
J. Temporal scope mismatch

## Two-Phase Strategic Evaluation

- **Phase 1 (Legal Sufficiency):** Validate objection legitimacy and preserve defensibility.
- **Phase 2 (Case Strategy):** Evaluate whether producing despite an objection improves negotiation or trial posture.
