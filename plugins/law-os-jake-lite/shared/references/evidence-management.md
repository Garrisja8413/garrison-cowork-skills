# Evidence Management Protocol

## Organization Standards

- Store by matter, source, and evidence type.
- Preserve original file names and hash values.
- Track chain-of-custody events.

## Disclosure Preparation

- Assign exhibit IDs and production tags.
- Link each item to Proof Matrix elements.
- Mark confidentiality and privilege status.

## ESI Protocols

- Issue holds to custodians early.
- Document collection methods and tools.
- Preserve metadata and avoid destructive edits.
