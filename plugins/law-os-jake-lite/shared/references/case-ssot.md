# Case Management SSOT

## Proof Matrix

Track every claim element against verified evidence rows:

- Claim element
- Burden owner
- Proof source (pack row ID)
- Admissibility status
- Gap / remediation task

## Defense Matrix

Maintain defense-aware strategy coverage:

- Defense theory
- Supporting facts alleged by defense
- Rebuttal facts in record
- Missing rebuttal evidence
- Assigned owner + due date

## Case Milestones

1. Intake complete + conflict cleared
2. Liability and coverage initial analysis
3. Preservation / LOR cycle complete
4. Discovery served + deficiency review
5. Expert and dispositive motion readiness
6. Pretrial order and trial prep complete
