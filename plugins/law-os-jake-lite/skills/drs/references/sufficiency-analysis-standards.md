# Discovery Response Sufficiency Analysis Standards — DRS Reference

**Version:** 1.0
**Applies to:** DRS (Discovery Response Sufficiency Analyzer)

## Purpose

DRS analyzes the *substance* of discovery responses — separate from objection
analysis — to determine whether answers actually respond to what was asked.
This reference defines the sufficiency standards, classification criteria, and
deficiency scoring methodology.

## Sufficiency Classification

### Category A — Fully Responsive
- Answer directly addresses all aspects of the question
- All responsive categories of information included
- Specificity matches the scope of the request
- No material omissions detected

### Category B — Partially Responsive
- Answer addresses the core question but omits one or more categories
- Specificity is adequate for some elements but vague for others
- Supplementation needed for completeness but no bad faith indicated

### Category C — Evasive
- Answer technically responds but avoids the substance of the question
- Uses qualifiers that effectively nullify the response ("to the best of...")
- Redirects to other discovery or documents without specifying which
- Provides information that is technically accurate but practically useless

### Category D — Non-Responsive
- Answer does not address the question asked
- Response discusses a different subject entirely
- "See prior response" where prior response is also non-responsive
- Boilerplate language substituted for a substantive answer

### Category E — Contradictory
- Answer conflicts with known facts from CFP
- Answer is internally inconsistent with other responses
- Answer contradicts documentary evidence already produced
- Answer contradicts deposition testimony

## Scoring Methodology

Each response receives a numerical sufficiency score:
- **10** — Fully responsive, no deficiencies
- **7-9** — Minor gaps or ambiguities, supplementation helpful but not required
- **4-6** — Material omissions, supplementation required
- **1-3** — Evasive or substantially non-responsive, enforcement action warranted
- **0** — Completely non-responsive or contradictory

## Cross-Reference Analysis

DRS cross-references each answer against:
- **CFP facts:** Does the answer contradict or confirm known facts?
- **PCM elements:** Does the answer fill or leave gaps in the proof matrix?
- **PDM elements:** Does the answer affect defense analysis?
- **Other responses:** Are answers internally consistent across related requests?

## Deficiency Reporting

Each deficiency finding must include:
- Request number and full text of the original request
- Full text of the response being analyzed
- Sufficiency classification (A through E)
- Numerical score with explanation
- Specific facts or elements affected (PCM/PDM cross-reference)
- Recommended enforcement action and supporting authority
