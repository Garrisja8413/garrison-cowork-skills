# CO CHECKLIST — Case Opening Status Tracker

## Purpose

A comprehensive, trackable checklist for every case opening. Use this
to verify that all required actions have been completed and to identify
any outstanding items.

## Master Checklist

### Phase 4: SETUP — Administrative Foundation (Day 1)

#### CMS & Records
- [ ] Case created in CMS — Case ID: ___
- [ ] IKP lead record linked to case (case_id populated)
- [ ] Case type and status set correctly
- [ ] Client contact record created (name, phone, email, address, DOB)
- [ ] Defendant contact record(s) created
- [ ] Defendant insurance contact record created
- [ ] Client auto insurance contact record created
- [ ] Client health insurance contact record created
- [ ] Medical provider contact records created (all known providers)
- [ ] Witness contact records created (if any)
- [ ] Responding officer records created

#### File Structure
- [ ] Digital file folder structure created
- [ ] Retainer and authorization documents filed in 04-Client-Documents
- [ ] Intake summary filed

#### Deadlines
- [ ] SOL docketed in master calendar — Date: ___
- [ ] SOL warnings set (1 year, 6 months, 90 days, 30 days, 14 days, 7 days)
- [ ] Internal filing deadline set (SOL minus 30 days)
- [ ] Tort Claims Notice deadline docketed (if govt) — Date: ___
- [ ] Civil Rights Notice deadline docketed (if govt/LE) — Date: ___
- [ ] CAL trigger events fired (SOL-EXPIRATION, CASE-SIGNED, custom if govt)

#### Team
- [ ] Lead Attorney assigned: ___
- [ ] Case Manager / Paralegal assigned: ___
- [ ] Legal Assistant assigned: ___
- [ ] Internal team notification sent with case summary

### Phase 5: DISPATCH — First 48 Hours

#### Letters of Representation
- [ ] Tort Claims Notice sent (if govt) — Date: ___ Method: ___
- [ ] LOR to Defendant Insurance sent — Date: ___ Method: ___
- [ ] LOR to Client Auto Insurance sent — Date: ___ Method: ___
- [ ] LOR to Client Health Insurance sent — Date: ___ Method: ___
- [ ] All LOR copies filed in 03-Correspondence/LOR/

#### Spoliation / Preservation (if applicable)
- [ ] Preservation targets identified (surveillance, black box, logs, etc.)
- [ ] Spoliation letter(s) sent — To: ___ Date: ___ Method: Certified mail
- [ ] Delivery receipt retained
- [ ] Copies filed in 03-Correspondence/

#### Official Reports
- [ ] Police/Crash Report requested — Agency: ___ Date: ___
- [ ] CAD/dispatch records requested (if applicable) — Date: ___
- [ ] 911 audio requested (if applicable) — Date: ___
- [ ] Fire/EMS report requested (if applicable) — Date: ___
- [ ] Animal control report requested (if applicable) — Date: ___
- [ ] Incident report requested from business (if applicable) — Date: ___

#### Initial Medical Records
- [ ] Ambulance records requested — Provider: ___ Date: ___
- [ ] ER/Hospital records requested — Facility: ___ Date: ___
- [ ] PCP records requested (if seen post-DOI) — Provider: ___ Date: ___
- [ ] Specialist records requested (if seen) — Provider(s): ___ Date: ___
- [ ] HIPAA releases on file for all providers

#### Client Immediate Needs
- [ ] Property damage claim opened (if MVC) — Carrier: ___ Claim#: ___
- [ ] Vehicle appraisal/estimate arranged
- [ ] Rental car secured (if needed)
- [ ] LOP referral provided (if client needs providers)
- [ ] MedPay/PIP claim opened (if applicable)

### Phase 6: HANDOFF — Pre-Litigation Transition

#### Introductory Call
- [ ] Introductory call completed — Date: ___ By: ___
- [ ] Team contacts provided to client
- [ ] LOR status confirmed with client
- [ ] Medical care rules reviewed
- [ ] Social media rules reviewed
- [ ] Communication rules reviewed
- [ ] Timeline expectations set
- [ ] Client questions addressed

#### Check-In Cadence
- [ ] Check-in frequency determined: Every ___ days
- [ ] First client check-in scheduled — Date: ___
- [ ] Recurring check-in task created in CMS
- [ ] 30-day medical records review task created
- [ ] 60-day attorney case review task created
- [ ] 90-day comprehensive assessment task created

#### Monitoring Plan
- [ ] Medical records tracking log initialized
- [ ] Medical bills tracking log initialized
- [ ] Lien tracking initialized (health ins, Medicare, Medicaid, etc.)
- [ ] Evidence preservation follow-up scheduled
- [ ] Police report follow-up scheduled (if not received)

## Escalation Triggers

Flag for immediate attorney attention if:

- [ ] SOL within 90 days and complaint not yet filed
- [ ] Tort Claims Notice deadline within 14 days and notice not sent
- [ ] Spoliation letter not sent within 7 days (evidence destruction risk)
- [ ] Client unreachable after 3 contact attempts
- [ ] Insurance company denying coverage or disputing liability
- [ ] New information changes case evaluation (new defendants, worse injuries, etc.)
- [ ] Client receiving direct contact from insurance despite LOR
