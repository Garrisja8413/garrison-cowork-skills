# Common Inference Patterns in PI Litigation

## Medical Malpractice Inferences

### Causation Chain
```
FACT: Patient's condition was [X] before treatment
FACT: Provider performed [procedure/treatment]
FACT: Patient's condition became [Y] after treatment
  → INFERENCE: Treatment caused the change in condition [MODERATE — needs expert]
  → INFERENCE: Provider's deviation from standard of care caused the harm [MODERATE — needs expert]
```

### Failure to Diagnose
```
FACT: Patient presented with [symptoms] on [date]
FACT: Provider did not order [test/imaging]
FACT: Condition was later diagnosed as [X] on [later date]
  → INFERENCE: Earlier testing would have revealed the condition sooner [MODERATE]
  → INFERENCE: Earlier diagnosis would have changed treatment and outcome [MODERATE — needs expert]
```

### Informed Consent
```
FACT: Consent form does not mention [specific risk]
FACT: Provider's notes do not document discussion of [risk]
FACT: [Risk] materialized and caused injury
  → INFERENCE: Provider did not disclose [risk] to patient [STRONG]
  → INFERENCE: Reasonable patient would have considered [risk] material [MODERATE]
  → INFERENCE: Reasonable patient would have declined treatment if informed [MODERATE]
```

### Records Deficiency / Spoliation
```
FACT: Medical records for [time period] are missing
FACT: Records were in defendant's exclusive control
FACT: Defendant was on notice of potential litigation
  → INFERENCE: Missing records contained information unfavorable to defendant [MODERATE]
```

## Premises Liability Inferences

### Notice
```
FACT: Hazardous condition existed for [duration]
FACT: Defendant's employees were present in the area
FACT: No evidence of inspection or remediation
  → INFERENCE: Defendant knew or should have known of the condition [STRONG if duration is substantial]
```

### Causation
```
FACT: Plaintiff was on defendant's property
FACT: Plaintiff fell at [location]
FACT: [Hazardous condition] existed at [location]
  → INFERENCE: [Hazardous condition] caused plaintiff's fall [STRONG if no alternative cause]
```

## Auto Collision Inferences

### Speed
```
FACT: Skid marks measured [X] feet
FACT: Road surface was [type] and [condition]
  → INFERENCE: Vehicle was traveling at approximately [speed] [MODERATE — needs expert/physics]
```

### Distraction
```
FACT: Defendant's phone records show text sent at [time]
FACT: Collision occurred at [time]
FACT: Times overlap within [margin]
  → INFERENCE: Defendant was using phone at time of collision [STRONG if overlap is tight]
```

## Insurance Bad Faith Inferences

### Unreasonable Denial
```
FACT: Insurer's own adjuster noted coverage appeared to apply
FACT: Insurer denied the claim citing [exclusion]
FACT: Internal memo discussed litigation cost vs. claim value
  → INFERENCE: Denial was not based on good faith coverage analysis [STRONG]
  → INFERENCE: Denial was motivated by economic considerations [MODERATE]
```

### Failure to Investigate
```
FACT: Claim was filed on [date]
FACT: First investigation activity occurred [X] days later
FACT: Industry standard is investigation within [Y] days
  → INFERENCE: Insurer failed to promptly investigate [STRONG if X >> Y]
```

## Contract/UPA Inferences

### Misrepresentation
```
FACT: Defendant advertised [feature/quality]
FACT: Product/service did not have [feature/quality]
FACT: Plaintiff purchased based on advertisement
  → INFERENCE: Plaintiff relied on the misrepresentation [STRONG]
  → INFERENCE: Defendant knew the representation was false [MODERATE — depends on evidence]
```

## Opponent Inference Weakness Patterns

### Speculation Flag
Opponent's inference is speculative when:
- "Could have" / "might have" / "possibly" language
- No record fact anchoring the inference
- Requires assuming facts not in evidence
- Multiple alternative explanations equally supported

### Contradiction Flag
Opponent's inference is contradicted when:
- Their own documents support the opposite conclusion
- Deposition testimony of their witness undercuts the inference
- Chronology is impossible or improbable

### Missing Link Flag
Opponent's inference chain has a missing link when:
- Jump from Fact to Conclusion skips necessary intermediate inference
- Expert testimony required but not provided
- Inference requires specialized knowledge presented as common sense

## Using Inference Patterns

1. **Start with the element you need to prove**
2. **Identify which CFP facts bear on that element**
3. **Match to the closest inference pattern above**
4. **Rate confidence based on fact strength and pattern fit**
5. **Identify gaps: what additional facts would strengthen the inference?**
6. **Flag where expert testimony is needed to bridge inferential steps**
